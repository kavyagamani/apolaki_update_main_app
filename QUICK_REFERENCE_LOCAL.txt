## 🧪 TEST LOGIN

### Via Browser
1. Open http://localhost:5173/login
2. Enter: admin@apolaki.com / admin123
3. Click Login
4. Should redirect to dashboard

### Via API/curl
```bash
curl -X POST http://localhost:3001/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@apolaki.com","password":"admin123"}'
```
Response should have: `"success": true` + JWT token

## 🔍 TROUBLESHOOTING

### Port Already in Use
```bash
lsof -i :3001 | grep LISTEN | awk '{print $2}' | xargs kill -9
lsof -i :5173 | grep LISTEN | awk '{print $2}' | xargs kill -9
```

### Database Not Running
```bash
# macOS with Homebrew
brew services start postgresql

# Check status
pg_isready -h localhost -p 5432
```

### Check Database Manually
```bash
psql -U apolaki_user -d apolaki_db -c "SELECT * FROM users;"
```

### Frontend Shows Blank Page
```bash
cd frontend
npm install
npm run dev
# Check browser console (F12) for errors
```

### Login Fails in Browser
1. Open DevTools (F12) → Network tab
2. Try login
3. Look for `/api/auth/login` request
4. Check response status (should be 200)
5. Check browser console for JS errors
